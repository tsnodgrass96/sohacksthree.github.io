Resources

The Essentials


  • Github.com

    Online Collaboration

    GitHub is a web-based hosting service for software development projects that use the Git revision control system. GitHub offers both paid plans for private repositories, and free accounts for open source projects. The site was launched in 2008 by Tom Preston-Werner, Chris Wanstrath, and PJ Hyett.


  • Teamtreehouse.com

    Video Programming Lessons

    Treehouse, a leading online education provider, seeks to help prospective coders become great coders by providing a rich library of over 600+ videos organized into specific learning tracks, along with interactive quizzes and code challenges. Treehouse provides deep and high quality training in HTML, CSS, Javascript, and User Experience.


  • Stackoverflow.com

    Q & A For Programming

    Stack Overflow is a privately held website. It features questions and answers on a wide range of topics in computer programming. The website serves as a platform for users to ask and answer questions, and, through membership and active participation, to vote questions and answers up or down and edit questions and answers in a fashion similar to a wiki or digg.


  • OverAPI.com

    Quick Reference

    OverAPI is a reference site for many of today's in demande programming languages and APIs. With hundreds of resources, it's mission is to "Get all the cheat sheets" and provide a consolidated database fo everyone to learn about various APIs and coding methods from.


  • HackathonHero.com

    General Hackathon Reference Site

    Hackathon hero is a website completely dedicated to hackathons and hackathon ideas. If you are stumped on what to make, Hackathon Hero has a database of common ideas for you to improve upon or completely reinvent the idea itself. Learn more at hackathonhero.com


  • Bentobox.io

    Programming Tutorial Site

    Bento is a comprehensive suite of resources to help you learn to code. It mainly provides free resources from around the web, including videos, interactive tutorials, and reading material. It is particularly good at teaching you what to learn in what order: by clicking on any of the boxes, it will highlight the boxes you should learn next. If in doubt, you should learn the resources in order from left to right, row by row.
